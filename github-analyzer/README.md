# GitHub Analyzer 🚀🏛️📦

> "Paste a GitHub URL. We analyze your entire codebase, show you every dependency, predict every risk, and when you're ready — we open a real Pull Request with the fixes."

## Tech Stack
- **Backend**: Python + FastAPI
- **Frontend**: Vanilla HTML/JS + CSS (No build step)
- **AI**: Risk prediction and fix generation

## Structure
- `backend/`: FastAPI server and services.
- `frontend/`: Lightweight UI for visualization.

## Setup
1. `cd backend`
2. `pip install -r requirements.txt`
3. `cp .env.example .env` (Add your API keys)
4. `python main.py`
